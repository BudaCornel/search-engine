# LocalSearch

A local file search engine that crawls, indexes, and searches files on your device using full-text search.

Built with **Java** and **PostgreSQL**.

## Features (Planned)

- Recursive file crawling with configurable ignore rules
- Full-text search using PostgreSQL's built-in FTS capabilities
- File previews in search results
- Incremental indexing (only re-index changed files)
